RECOMMENDATIONS
================================================================================

1. WRONG TEXT (id=1155): Delete or mark as invalid. This is the Decretals
   of Gregory IX, not Gregory the Great's letters.

2. CONCATENATED RECORDS: For each concatenated record:
   a. Identify the boundaries of the FIRST letter in the concatenation
   b. Truncate latin_text to contain only the first letter
   c. Strip editorial apparatus (variant readings, footnotes)
   d. The existing modern_english translations are ACCURATE for the first
      letter and do NOT need re-translation
   e. Optionally, create new records for the additional letters found in
      each concatenation (but check for existing records first to avoid
      further duplication)

3. EDITORIAL APPARATUS: For single-letter records with heavy apparatus:
   a. Strip variant readings, footnotes, and editorial commentary
   b. Retain only the letter text itself
   c. This is lower priority than fixing concatenation issues

4. DUPLICATE PAIRS: For each duplicate pair:
   a. Determine which ref_id is canonical (prefer MGH numbering)
   b. Delete or merge the duplicate record
   c. The PL77 book numbering (books 2-7) often maps multiple PL77
      numbers to the same underlying letter

5. NO TRANSLATION FIXES NEEDED: All examined translations for concatenated
   records accurately translate the first letter in the concatenation.
   They are short because they cover only one letter, not the full
   concatenated text. This is correct behavior.
